# Zohair Omar — Portfolio

A VHS/DVD-menu themed personal portfolio website.

**Live:** [zohairomar.netlify.app](https://zohairomar.netlify.app/)

## Built With

- Next.js, TypeScript, Tailwind CSS
- LaTeX resume auto-parsed to JSON
- VHS effects: scanlines, CRT flicker, film grain, retro themes

## Sections

- **About Me** — who I am
- **Projects** — things I've built
- **Experience & Education** — where I've worked and studied
- **Bonus Features** — lessons learned and bloopers
- **Contact** — get in touch / download resume
